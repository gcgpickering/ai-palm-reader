# AI Palm Reader 🔮

An AI-powered palmistry reading platform built as a fully automated WhatsApp bot using **n8n workflow automation**. Users text a WhatsApp number, upload photos of their palms, and receive personalized palm readings generated by **OpenAI Vision (GPT-4o)** — complete with HTML reports, subscription tiers via **Stripe**, and ongoing follow-up insights.

## How It Works

### 1. Onboarding & Consent
- User texts **"Read My Palms"** to the WhatsApp number
- Bot collects: first name, date of birth, time of birth, birthplace
- Location is geocoded automatically via Google Maps API
- **GDPR-compliant consent** is collected and logged to an audit table before any analysis

### 2. Palm Image Analysis
- User uploads photos of both palms (right and left)
- Images are stored in Supabase Storage
- **OpenAI Vision (GPT-4o)** analyzes each palm independently:
  - Heart line, head line, life line, fate line
  - Mount analysis, finger proportions
  - Skin texture and markings

### 3. Reading Generation
- AI synthesizes both palm analyses with birth data into a comprehensive reading
- Reading is rendered as a styled **HTML report** and uploaded to Supabase Storage
- User receives a direct link to their personalized reading

### 4. Subscription Tiers (Stripe Integration)

| Tier | Features |
|------|----------|
| **Free** | Initial palm reading |
| **Standard** | Ongoing insights, weekly transit alerts |
| **Annual** | Everything in Standard + priority |
| **Family** | Multi-person readings |

### 5. Premium Add-on Reports

| Add-on | Description |
|--------|-------------|
| **Compatibility Bridge** | Relationship compatibility analysis |
| **Vocation Blueprint** | Career guidance based on palm features |
| **Child Legacy** | Parent-child palmistry insights |

### 6. Ongoing Engagement
- **Hourly follow-up checks** — personalized messages based on life struggle classification
- **Weekly planetary transit alerts** — AI-generated astrological insights for subscribers
- **Quarterly deep-dive reports** — comprehensive re-analysis for long-term subscribers
- **Microscan feature** — quick re-scan of palms to track changes over time

## Workflow Nodes (~80+ nodes)

The n8n workflow contains **80+ interconnected nodes** handling:

- **Inbound message processing** — Twilio webhook receives WhatsApp messages
- **State machine** — JavaScript code node tracks user progress through a multi-step conversation
- **Intent classification** — OpenAI classifies user messages (purchase, cancel, scan, status, help, etc.)
- **Image analysis pipeline** — Dual-palm OpenAI Vision analysis with structured output
- **Stripe payment flow** — Checkout session creation, webhook processing, subscription management
- **Report generation** — HTML template rendering with AI content, uploaded to cloud storage
- **Scheduled automations** — Hourly follow-ups, weekly transits, quarterly deep-dives

## Setup

### Prerequisites
- [n8n](https://n8n.io) instance (self-hosted or cloud)
- Twilio account with WhatsApp-enabled number
- OpenAI API key (GPT-4o access)
- Supabase project (database + storage)
- Stripe account (test or live)
- Google Maps Geocoding API key

### Installation

1. **Import the workflow** into your n8n instance:
   - Open n8n → Workflows → Import from File → select `workflow.json`

2. **Configure credentials** in n8n:
   - **Twilio** — Account SID, Auth Token, WhatsApp number
   - **OpenAI** — API key with GPT-4o access
   - **PostgreSQL** — Supabase connection string (or direct Postgres)
   - **Stripe** — Secret key for payment processing

3. **Set up the database** — Create the following tables in your PostgreSQL/Supabase instance:
   - `users` — User profiles, consent status, subscription tier
   - `readings` — Palm analysis results and report URLs
   - `consent_audit` — GDPR consent log
   - `addons` — Purchased add-on report tracking

4. **Configure webhooks**:
   - Set your Twilio WhatsApp webhook to: `https://your-n8n-url/webhook/inbound-message`
   - Set your Stripe webhook to: `https://your-n8n-url/webhook/stripe`

5. **Activate the workflow** and text "Read My Palms" to your WhatsApp number
